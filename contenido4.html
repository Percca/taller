Condicionales en Programación

Los condicionales son estructuras fundamentales en programación que permiten ejecutar diferentes bloques de código según si una condición es verdadera (true) o falsa (false).

Condicional IF

temperatura = 30
if temperatura > 25:
print("Hace calor")
# Se ejecuta solo si temperatura > 25
print("Recomiendo beber agua")
print("Este mensaje siempre aparece")


Condicional ELSE-IF

edad = 17
if edad >= 18:
print("Puedes entrar al club")
print("Disfruta la fiesta")
else:
print("Acceso denegado")
print("Debes ser mayor de edad")
print("Fin de la verificación")


Condicionales anidados

usuario = "admin"
contraseña = "1234"

if usuario == "admin":
if contraseña == "1234":
print("Acceso concedido")
else:
print("Contraseña incorrecta")
else:
print("Usuario no reconocido")


Condicional ternario

valor = expresión_verdadera if condición else expresión_falsa
