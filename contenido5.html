Ciclos en Programación

Los ciclos (o bucles) son estructuras fundamentales que permiten repetir bloques de código bajo ciertas condiciones. Son esenciales para automatizar tareas repetitivas y procesar colecciones de datos.

Ciclo While

#Ejecuta un bloque mientras una condición sea verdadera.

contador = 0
while contador < 5:
print(f"Iteración {contador}")
contador += 1 # Incremento para evitar ciclo infinito


Ciclo For

#Itera sobre una secuencia (lista, rango, cadena, etc.).

for i in range(3): # 0, 1, 2
print(f"Número: {i}")
